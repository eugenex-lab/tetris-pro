# Flutter Game Design Skill

## Purpose
Assist in designing and building mobile games using Flutter and the Flame engine.

---

## When To Use
Use this skill when:
- Designing Flutter game architecture
- Creating 2D mobile games
- Implementing animations and physics
- Building game UI and controls
- Creating scoring and level systems
- Adding monetization to games
- Designing casual cozy game interfaces
- Creating family-friendly puzzle and arcade games

---

## Recommended Tools

### Game Engine
- Flame
- Flame Forge2D

### Animation
- Rive
- Lottie
- Flutter animation controllers

### Audio
- flame_audio

### Monetization
- google_mobile_ads

---

## Project Setup

### Create Game Project
```bash
flutter create my_game
cd my_game
flutter pub add flame
flutter pub add flame_audio
